IBM ILOG CPLEX Optimization Studio 20.1

IBM® ILOG® CPLEX® Optimization Studio est le moyen le plus rapide de créer des modèles d'optimisation efficaces et des applications de pointe permettant de traiter les multiples problèmes de planification et d'ordonnancement. Avec son environnement de développement intégré, son langage de modélisation descriptive, ses outils intégrés et ses solveurs avancés, traitant les problèmes de programmation mathématique et les problèmes de programmation par contraintes, il prend en charge le processus complet de développement et de résolution des modèles.

CPLEX Optimization Studio comprend plusieurs composants. Certains composants ne sont pas disponibles sur toutes les plateformes. Pour en savoir plus, rendez-vous sur la page relative à la configuration système requise qui est décrite en détail ci-dessous. Chacun des composants est placé dans un sous-dossier (ou sous-répertoire pour Linux) de l'emplacement d'installation. Les API et les connecteurs pour chaque composant se trouvent dans le sous-dossier (sous-répertoire) de ce composant.

  • opl : environnement de modélisation OPL et environnement de développement intégré (IDE)
  • cplex : optimiseurs CPLEX incluant les optimiseurs de simplexe, de barrière et mixte en nombres entiers
  • cpoptimizer : optimiseur PPC (CP Optimizer), qui utilise la technologie de programmation par contraintes
  • concert : couche de modélisation C++ à utiliser avec les optimiseurs
  • python : couche de modélisation Python DOcplex, pour les optimiseurs CPLEX et CP

Autres répertoires :

  • doc : version HTML de la documentation
  • jre : environnement d'exécution Java™ utilisé pour les processus d'installation et de désinstallation (sur les plateformes dépourvues d'IDE OPL)
  • license : documents de contrat de licence
  • swidtag : informations d'inventaire
  • Uninstall : informations nécessaires pour désinstaller le produit, et journaux d'installation

Installation sous Windows

Sous Windows, l'emplacement d'installation par défaut d'IBM ILOG CPLEX Optimization Studio est :
C:\Program Files\IBM\ILOG\CPLEX_Studio201

L'emplacement d'installation de l'édition Community est :
C:\Program Files\IBM\ILOG\CPLEX_Studio_Community201

Lors de l'installation, vous avez peut-être sélectionné un emplacement différent, qui sera désigné par [rép_install]) dans le reste de ce document.

Important

Ce produit nécessite l'installation du package redistribuable Microsoft Visual C++ 2015-2019 (x64). Vous pouvez télécharger ce package ici : Microsoft Visual C++ 2015-2019 Redistributable Package (x64).

CPLEX Optimization Studio est livré sur la plateforme Windows comme un unique port. Nommé x64_windows_msvc14, ce port est compatible avec Visual Studio 2015, Visual Studio 2017 et Visual Studio 2019. Il a été créé et compilé avec Visual Studio 2015. Pour plus de détails sur la compatibilité des différentes versions de Visual Studio entre elles, consultez C++ Binary Compatibility between Visual Studio 2015 and Visual Studio 2019.

IDE

Vous pouvez lancer l'IDE en utilisant une entrée du menu Démarrer :

Démarrer > Tous les programmes > CPLEX Optimization Studio 20.1.0 > IDE de CPLEX Studio 20.1.0

Pour l'édition Community :

Démarrer > Tous les programmes > CPLEX Optimization Studio Community Edition 20.1.0 > IDE de CPLEX Studio 20.1.0

Installation sous Linux

Le programme d'installation pour Linux est un fichier exécutable. Pour lancer une installation, vous devez donner à ce fichier un droit d'exécution et vous devez l'exécuter à partir d'une ligne de commande. Par exemple, lancez la commande ./<nom_programme_installation>.bin où <nom_programme_installation> est le nom du programme d'installation. Vous pouvez donner l'autorisation d'exécution à un fichier avec la commande chmod u+x <nom_programme_installation>.bin.

Sous Linux, l'emplacement d'installation par défaut d'IBM ILOG CPLEX Optimization Studio est :
/opt/ibm/ILOG/CPLEX_Studio201

L'emplacement d'installation de l'édition Community est :
/opt/ibm/ILOG/CPLEX_StudioCommunity201

IDE

Vous pouvez lancer l'IDE depuis un terminal Linux.
1. Ouvrez une fenêtre de terminal
2. Passez dans le répertoire [rép_install]/opl/oplide
3. Entrez la commande ./oplide

Vous pouvez également entrer le chemin d'accès absolu à partir de n'importe quel emplacement du terminal :
[rép_install]/opl/oplide/oplide
