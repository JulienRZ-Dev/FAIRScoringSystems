frac

OPL function for the fractional part

Purpose

OPL function to return the fractional part.

type 
float

Syntax

frac(float f)

Description

Returns the fractional part of a floating-point value. You can use this function within IBM ILOG Script statements by specifying the OPL namespace:


( Opl.xxx() )

Example

float a=-2.125;
float b=3.5;

float fracofa=frac(a);
float fracofb=frac(b);

execute
{
   writeln("frac(",a,") gives ",fracofa);
   writeln("frac(",b,") gives ",fracofb); 
}

Result

frac(-2.125) gives 0.125
frac(3.5) gives 0.5
