cumulFunctionValue

OPL function (scheduling) to return the value of a cumulative function.

Purpose

OPL function (scheduling) to return the value of a cumulative function.

context type
Model files (.mod) — postprocessing 
int

Syntax

cumulFunctionValue(f, i);

Where:

int i;
cumulFunction f;

Description

This function returns the value of cumulative function f at time t in the solution. This function is available when a solution exists.

Example

A basic program using cumulFunctionValue to show the effects of using pulse with cumulFunction:

using CP;

dvar interval it;
cumulFunction c=pulse(it,10);

subject to
{
 startOf(it)==2;
 endOf(it)==4;
}

int v[i in 0..5]=cumulFunctionValue(c,i);
execute
{
  writeln(v);
}

assert forall(i in 2..3) v[i]==10;
assert forall(i in 0..5: i!=2 && i!=3) v[i]==0;
