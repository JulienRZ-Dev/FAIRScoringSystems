ord

OPL function to return the position of an element

Purpose

OPL function to return the position of an element in a set.

type 
int or a tuple type

Syntax

int ord({int},int)
int ord({float},float)
int ord({string},string)
int ord({<tuple type>},<tuple type>)

Description

Returns the position of an element in a set. Given a set S, the function ord(S,e) returns the position of e, an element of S, in S. Positions start at 0 and ord(S,e) produces an execution error if e is not in S.

You can use this function within IBM ILOG Script statements by specifying the OPL namespace:


( Opl.xxx() )

Example

When applied to the set:


{string} Days {“Monday”, “Tuesday”, “Wednesday”, “Thursday”, “Friday”, 
“Saturday”, “Sunday”};

The function ord(Days, “Tuesday”) returns 1 and ord(Days, “Sunday”) returns 6.
