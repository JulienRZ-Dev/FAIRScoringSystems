matchAt

OPL function to get the first position of a substring

Purpose

OPL function to get the first position of a substring in a string.

type 
int

Syntax

matchAt(string,substring)

Description

Gets the first position of a substring in a string. If the substring is not found, the function returns -1.

You can use this function within IBM ILOG Script statements by specifying the OPL namespace:


( Opl.xxx() )

Example

string s="Monday";
string subs1="day";
string subs2="days";

int pos1=matchAt(s,subs1);
int pos2=matchAt(s,subs2);

execute
{
   writeln("matchAt(\"",s,"\",\"",subs1,"\") gives ",pos1);
   writeln("matchAt(\"",s,"\",\"",subs2,"\") gives ",pos2);
}

Result

matchAt("Monday","day") gives 3
matchAt("Monday","days") gives -1
