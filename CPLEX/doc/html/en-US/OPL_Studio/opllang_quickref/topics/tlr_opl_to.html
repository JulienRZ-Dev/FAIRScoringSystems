to

OPL keyword for database and spreadsheet writing

Purpose

OPL keyword used in association with DBUpdate and SheetWrite.

context
Data files (.dat)

Syntax

Publish: "DBExecute" '(' Id String ')'
       | Id "to" "DBUpdate" '(' Id String ')'
       | Id "to" "SheetWrite" '(' Id String ')'

Description

This keyword is used in data files in association with the DBUpdate and SheetWrite keywords to write the results of a model solve back to the database or spreadsheet.

Example

See the oilDB and oilSheet examples in the current distribution.
