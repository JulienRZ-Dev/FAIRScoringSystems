ftoi

OPL function to convert a float to an integer

Purpose

OPL function to convert a float representing an integer to an integer represented as an integer.

type 
int

Syntax

ftoi(float f)

Description

The function ftoi (float to integer) can be used to convert an integer represented as a floating-point value to an integer represented as an integer. You can use this function within IBM ILOG Script statements by specifying the OPL namespace:


( Opl.xxx() )

In constraint programming models,.ftoi should be read as ftoi(round). As a consequence, this code:

using CP;
dvar int x;

maximize x;
  subject to 
{
  ftoi(x+0.1)==1;
}

gives

x==1

even though ftoi(1.1) does not exist.

Example

Model

int i = ftoi(round(4.7));

execute
{
   writeln("ftoi(round(4.7)) gives ",i);
}

Result

ftoi(round(4.7)) gives 5

Note that the declaration:


int i = round(4.7);

produces a semantic error. The result of function ftoi is undefined when the integer cannot be represented exactly in type int.
