minl

OPL function to return the minimal value

Purpose

OPL function to return the minimal value from a list of integers or floats.

type 
int or float

Syntax

int minl(int,...)
float minl(float,...)

Description

Returns the minimal value from a list of integer or float values. You can use this function within IBM ILOG Script statements by specifying the OPL namespace:


( Opl.xxx() )

Example

int minofs1=minl(1, 2 ,5 ,10, 7 ,8);
float minofs2=minl(2.5 ,1.522e2, -10.2);



execute
{
   writeln("minl(1, 2 ,5 ,10, 7 ,8) gives ",minofs1);
   writeln("minl(2.5 ,1.522e2, -10.2) gives ",minofs2);
}

Result

minl(1, 2 ,5 ,10, 7 ,8) gives 1
minl(2.5 ,1.522e2, -10.2) gives -10.2
