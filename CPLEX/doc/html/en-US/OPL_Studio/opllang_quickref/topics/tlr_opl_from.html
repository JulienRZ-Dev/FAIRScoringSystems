from

OPL keyword for database and spreadsheet reading.

Purpose

OPL keyword used in association with DBRead and SheetRead.

context
Data files (.dat)

Syntax

Read: Id "from" "DBRead" '(' Id String ')' PlaceHolders_opt
    | Id "from" "SheetRead" '(' Id String ')'

Description

This keyword is used in data files in association with the DBRead and SheetRead keywords to read data from a database or spreadsheet.

Example

See the oilDB, oilDB2, and oilSheet examples in the current distribution.
