Search control/Log control

Describes the general options that enable you to control the log tracing the search process.

Click Constraint Programming>Search control>Log control in the tree structure in the left pane of the editor, as shown in the following screen capture.

The following table describes Search control/Log control options.

Table 1. CP Search control/Log control options 
Name Description Default Value
Log period The CP Optimizer search log includes information that is displayed periodically. This parameter controls how often that information is displayed. A value of k means that the log is displayed every k branches (search decisions). 1000
Log verbosity Determines the verbosity of the search log. “Quiet” displays no information. In the current version of CP Optimizer, levels “Terse”, “Normal”, and “Verbose” display the same information. Normal
