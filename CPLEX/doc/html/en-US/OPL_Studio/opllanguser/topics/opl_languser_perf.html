Performance and memory usage

You can improve the modeling and the solving time of your models, and the ability to find satisfactory solutions to a problem expressed in a model.

  • Performance tips
    Contains a checklist of modeling best practices.
  • Memory usage
    Explains how OPL uses and allocates memory, and suggests actions to improve memory usage, mostly for data structures, object termination, engine parameters, and oplrun.
  • Multi-threading
    An example is provided to illustrate how to use multi-threading to solve models more quickly.
