Changes to Watson Machine Learning (WML) connector

This latest version of CPLEX Optimization Studio 20.1.0 includes updates to the WML connector. Also, the connector to DOcplexcloud, deprecated in 12.10.0, has been removed.

Watson Machine Learning credentials

The revised Watson Machine Learning (WML) connector API requires some changes to credentials. The new credentials are Space Id and Bucket. These credentials are created in IBM Cloud before setting up the connector to WML.

For details on how to use this connector, please refer to Solving OPL models on the cloud.

Connector to DOcplexcloud removed

The connector to Decision Optimization on Cloud was deprecated in CPLEX Optimization Studio 12.10.

In 20.1.0, the DOcplexcloud connector has been removed. Please use the connector to Watson Machine Learning introduced in 12.10.0.
