Release notes for IDE and OPL 20.1.0

These release notes highlight changes in the IDE and OPL 20.1.0. Please review these notes before using IDE and OPL 20.1.0.

  • New database support in OPL
    This latest version of CPLEX Optimization Studio 20.1.0 has increased database support in OPL. There are prebuilt database connection types for JDBCConnection, CSVConnection, ODBCConnection, MySQLConnection, and SQLiteConnection. In addition, you can build a custom database connection type. There are also several examples provided to demonstrate this functionality.
  • Changes to Watson Machine Learning (WML) connector
    This latest version of CPLEX Optimization Studio 20.1.0 includes updates to the WML connector. Also, the connector to DOcplexcloud, deprecated in 12.10.0, has been removed.
  • Other new features in OPL
    These release notes highlight other changes in 20.1.0 to the Optimization Programming Language (OPL), including changes to the OPL interfaces and the IBM ILOG Script for OPL.
