Using specialized constraints and tuples: scheduling teams

In this lesson, you will learn how to use CP Optimizer to find a solution to a sports scheduling problem.

  • Overview
    A solution to the team scheduling problem can be determined using CP Optimizer and the three-stage method of describe, model and solve.
  • Describe
    The first stage in finding a solution to the sports scheduling problem is to describe the problem using natural language.
  • Model
    Once the sports scheduling problem has been described using natural language, you then use Concert Technology classes to model the constraint programming problem.
  • Solve
    Once the sports scheduling problem has been described and modeled, you use CP Optimizer classes and functions to solve the constraint programming problem.
  • Review exercises
    Complete these exercises to review what you have learned in the tutorial. Suggested answers are provided.
  • Complete program
    The location of the complete sports scheduling program and the results are provided.
