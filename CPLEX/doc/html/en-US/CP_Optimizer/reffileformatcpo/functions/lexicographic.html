lexicographic

Returns a constraint which maintains two arrays to be lexicographically ordered.

Syntax


constraint lexicographic(intExprArray x, intExprArray y)

Parameters

  • x: an array of integer expressions
  • y: an array of integer expressions

Description

The lexicographic function returns a constraint which maintains two arrays to be lexicographically ordered.

More specifically, lexicographic(x, y) maintains that x is less than or equal to y in the lexicographical sense of the term. This means that either both arrays are equal or that there exists i < size(x) such that for all j < i, x[j] = y[j] and x[i] < y[i].

Example

lexicographic is often used for symmetry breaking purposes. Assuming your model includes a matrix where rows can be interchanged, it is possible to reduce the problem search space by enforcing a lexicographic order between rows: 


x1 = intVar(0..10);
x2 = intVar(0..10);
x3 = intVar(0..10);
y1 = intVar(0..10);
y2 = intVar(0..10);
y3 = intVar(0..10);
lexicographic([x1, x2, x3], [y1, y2, y3]);

Naturally, if your problem's matrix also has permutable columns, you can use the same approach on them.

Requirements

Arrays x and y must have the same size.

Notes

Strict lexicographic order between arrays of length N can be enforced using the strictLexicographic function.
