startEval

Evaluates segmentedFunction at the start of an interval variable.

Syntax


floatExpr startEval(intervalVar interval, segmentedFunction function, float absentValue = 0)

Parameters

  • interval: interval variable.
  • function: function to evaluate.
  • absentValue: value to return if interval variable interval is absent.

Description

Evaluates function at the start of interval variable interval. However if interval is absent then it does not have any defined start and absentValue is returned.

Example

Task T is urgent: there is a cost associated with starting it late which is expressed by segmentedFunction:

In the picture above, task T starts at 12 and the cost is therefore 5.

The cost can be computed using startEval as: 


function = segmentedFunction((0, 0), (10, 5, 0), (15, 5, 3));
T = intervalVar();
cost = startEval(T, function);
