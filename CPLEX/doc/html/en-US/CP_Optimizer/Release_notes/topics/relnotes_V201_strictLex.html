New constraint to sort in strict lexicographic order

Provides details of the new CP Optimizer constraint StrictLexicographic.

This constraint states that the first array of variables is strictly less than the second array of variables in the lexicographic order. Please consult the reference manuals of the CP Optimizer APIs for more detailed information. This constraint is available in the following APIs:

  • C++ at the modeling level: IloStrictLexicographic
  • C++ at the advanced extensions level: IlcStrictLexicographic
  • Java: IloCP.strictLexicographic
  • C#: CP.StrictLexicographic
  • CPO File Format: strictLexicographic
