DOcplex Python for CP Optimizer users

Provides details of new features and changes for the DOcplex Python modeling API for CP Optimizer users.

DOcplex is an object-oriented modeling Python API that can be used to access the CP Optimizer constraint programming engine. For information about the release notes of DOcplex, see Release notes for DOcplex Python API.

For more general information about using the DOcplex Python API with the CP Optimizer engines, see DOcplex Python Modeling API.
