Release notes for CP Optimizer 20.1.0

These release notes highlight improvements and new features of CP Optimizer in IBM® ILOG® CPLEX® Optimization Studio 20.1.0.

  • New CP Optimizer black-box optimization feature
    A black-box expression is a numerical expression for which the closed form is not known or cannot be formulated using the classical expressions available with CP Optimizer. This is now available in the C++ API.
  • CP Optimizer black-box expressions
    Provides a more detailed overview of CP Optimizer black-box expressions.
  • New constraint to sort in strict lexicographic order
    Provides details of the new CP Optimizer constraint StrictLexicographic.

    Provides details of new features and changes for C++ users.
  • Java users
    Provides details of new features and changes for Java users, as well as some limitations.
  • C#/.NET users
