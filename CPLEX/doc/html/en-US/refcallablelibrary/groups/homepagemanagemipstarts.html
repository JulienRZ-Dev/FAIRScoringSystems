Managing MIP Starts in the CPLEX Callable Library (C API)

These routines manage MIP starts, that is, the means for resuming optimization of a mixed integer program at a point where previous optimization stopped. MIP starts are also known as warm starts.

  • CPXXaddmipstarts and CPXaddmipstarts
    The routine CPXXaddmipstarts/CPXaddmipstarts adds multiple MIP starts to a CPLEX problem object of type CPXPROB_MILP, CPXPROB_MIQP, or CPXPROB_MIQCP.
  • CPXXchgmipstarts and CPXchgmipstarts
    The routine CPXXchgmipstarts/CPXchgmipstarts modifies or extends multiple MIP starts.
  • CPXXdelmipstarts and CPXdelmipstarts
    The routine CPXXdelmipstarts/CPXdelmipstarts deletes a range MIP starts.
  • CPXXdelsetmipstarts and CPXdelsetmipstarts
    The routine CPXXdelsetmipstarts/CPXdelsetmipstarts deletes a set of MIP starts.
  • CPXXgetmipstartindex and CPXgetmipstartindex
    The routine CPXXgetmipstartindex/CPXgetmipstartindex searches for the index number of the specified MIP start in a CPLEX problem object.
  • CPXXgetmipstartname and CPXgetmipstartname
    Accesses a range of names of MIP starts in a CPLEX problem object.
  • CPXXgetmipstarts and CPXgetmipstarts
    The routine CPXXgetmipstarts/CPXgetmipstarts accesses a range of MIP starts of a CPLEX problem object.
  • CPXXgetnummipstarts and CPXgetnummipstarts
    The routine CPXXgetnummipstarts/CPXgetnummipstarts accesses the number of MIP starts in the CPLEX problem object.
  • CPXXreadcopymipstarts and CPXreadcopymipstarts
    The routine CPXXreadcopymipstarts/CPXreadcopymipstarts reads a file in the format MST and copies the information of all the MIP starts contained in that file into a CPLEX problem object.
  • CPXXrefinemipstartconflict and CPXrefinemipstartconflict
    This routine is deprecated. Use CPXXrefinemipstartconflictext/CPXrefinemipstartconflictext instead.
  • CPXXrefinemipstartconflictext and CPXrefinemipstartconflictext
    The routine CPXXrefinemipstartconflictext/CPXrefinemipstartconflictext refines a conflict of a model inferred from a MIP start, where a MIP start consists of pairs of variables and values and where the model may include quadratic constraints, indicator constraints, or special ordered sets, as well as linear constraints and bounds.
  • CPXXwritemipstarts and CPXwritemipstarts
    The routine CPXXwritemipstarts/CPXwritemipstarts writes a range of MIP starts of a CPLEX problem object to a file in MST format.
