H, I, J, K, L

  • CPXXhybbaropt and CPXhybbaropt
    The routine CPXXhybbaropt/CPXhybbaropt may be used, at any time after a linear program has been created via a call to CPXXcreateprob/CPXcreateprob, to find a solution to that problem.
  • CPXXhybnetopt and CPXhybnetopt
    The routine CPXXhybnetopt/CPXhybnetopt, given a linear program that has been created via a call to CPXXcreateprob/CPXcreateprob, extracts an embedded network, uses the CPLEX Network Optimizer to attempt to obtain an optimal basis to the network, and optimizes the entire linear program using one of the CPLEX simplex methods.
  • CPXXindconstrslackfromx and CPXindconstrslackfromx
    The routine CPXXindconstrslackfromx/CPXindconstrslackfromx accesses the slack values for the indicator constraints corresponding to a given vector x.
  • CPXXinfodblparam and CPXinfodblparam
    The routine CPXXinfodblparam/CPXinfodblparam obtains the default, minimum, and maximum values of a CPLEX parameter of type double.
  • CPXXinfointparam and CPXinfointparam
    The routine CPXXinfointparam/CPXinfointparam obtains the default, minimum, and maximum values of a CPLEX parameter of type CPXINT.
  • CPXXinfolongparam and CPXinfolongparam
    The routine CPXXinfolongparam/CPXinfolongparam obtains the default, minimum, and maximum values of a CPLEX parameter of type CPXLONG.
  • CPXXinfostrparam and CPXinfostrparam
    The routine CPXXinfostrparam/CPXinfostrparam obtains the default value of a CPLEX string parameter.
  • CPXXkilldnorms and CPXkilldnorms
    The routine CPXXkilldnorms/CPXkilldnorms deletes any dual steepest-edge norms that have been retained relative to an active basis.
  • CPXXkillpnorms and CPXkillpnorms
    The routine CPXXkillpnorms/CPXkillpnorms deletes any primal steepest-edge norms that have been retained relative to an active basis.
  • CPXXlpopt and CPXlpopt
    At any time after a linear program has been created by a call to CPXXcreateprob/CPXcreateprob, the routine CPXXlpopt/CPXlpopt finds a solution to that problem using one of the CPLEX linear optimizers.
  • CPXXlprewrite and CPXlprewrite
    This function is deprecated. Use CPXXwriteprob and CPXwriteprob instead.
  • CPXXlpwrite and CPXlpwrite
    This function is deprecated. Use CPXXwriteprob and CPXwriteprob instead.
