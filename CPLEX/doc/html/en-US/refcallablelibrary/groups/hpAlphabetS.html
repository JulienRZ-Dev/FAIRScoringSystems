S

  • CPXXsavwrite and CPXsavwrite
    This function is deprecated. Use CPXXwriteprob and CPXwriteprob instead.
  • CPXXserializercreate and CPXserializercreate
    This routine creates a new serializer.
  • CPXXserializerdestroy and CPXserializerdestroy
    Destroys a serializer that was created by CPXXserializercreate/CPXserializercreate.
  • CPXXserializerlength and CPXserializerlength
    Queries the size of the memory block currently used by the serializer ser.
  • CPXXserializerpayload and CPXserializerpayload
    Queries the internal memory buffer used by ser.
  • CPXXsetbranchcallbackfunc and CPXsetbranchcallbackfunc
    Sets and modifies the user-written legacy callback routine to be called after a branch has been selected but before the branch is carried out during MIP optimization.
  • CPXXsetbranchnosolncallbackfunc and CPXsetbranchnosolncallbackfunc
    The routine CPXXsetbranchnosolncallbackfunc/CPXsetbranchnosolncallbackfunc sets the callback function that will be called instead of the branch callback when there is a failure due to such situations as an iteration limit being reached, unboundedness being detected, numeric difficulties being encountered, while the node LP is being solved.
  • CPXXsetdblannotations and CPXsetdblannotations
    The routine CPXXsetdblannotations/CPXsetdblannotations sets the annotation values for a set of modeling objects in the CPLEX problem object.
  • CPXXsetdblparam and CPXsetdblparam
    The routine CPXXsetdblparam/CPXsetdblparam sets the value of a CPLEX parameter of type double.
  • CPXXsetdefaults and CPXsetdefaults
    The routine CPXXsetdefaults/CPXsetdefaults resets all CPLEX parameters and settings to default values (with the exception of the log file).
  • CPXXsetdeletenodecallbackfunc and CPXsetdeletenodecallbackfunc
    The routine CPXXsetdeletenodecallbackfunc/CPXsetdeletenodecallbackfunc sets and modifies the user-written callback to be called during MIP optimization when a node is to be deleted.
  • CPXXsetheuristiccallbackfunc and CPXsetheuristiccallbackfunc
    The routine CPXXsetheuristiccallbackfunc/CPXsetheuristiccallbackfunc sets or modifies the user-written callback to be called by CPLEX during MIP optimization after the subproblem has been solved to optimality.
  • CPXXsetincumbentcallbackfunc and CPXsetincumbentcallbackfunc
    The routine CPXXsetincumbentcallbackfunc/CPXsetincumbentcallbackfunc sets and modifies the user-written callback routine to be called when an integer solution has been found but before this solution replaces the incumbent.
  • CPXXsetinfocallbackfunc and CPXsetinfocallbackfunc
    The routine CPXXsetinfocallbackfunc/CPXsetinfocallbackfunc sets the user-written callback routine that CPLEX calls regularly during the optimization of a mixed integer program and during certain cut generation routines.
  • CPXXsetintparam and CPXsetintparam
    The routine CPXXsetintparam/CPXsetintparam sets the value of a CPLEX parameter of type CPXINT.
  • CPXXsetlazyconstraintcallbackfunc and CPXsetlazyconstraintcallbackfunc
    The routine CPXXsetlazyconstraintcallbackfunc/CPXsetlazyconstraintcallbackfunc sets and modifies the user-written callback for adding lazy constraints.
  • CPXXsetlogfilename and CPXsetlogfilename
    Sets and opens the log file.
  • CPXXsetlongannotations and CPXsetlongannotations
    The routine CPXXsetlongannotations/CPXsetlongannotations sets the annotation values for a set of modeling objects in the CPLEX problem object.
  • CPXXsetlongparam and CPXsetlongparam
    The routine CPXXsetlongparam/CPXsetlongparam sets the value of a CPLEX parameter of type CPXLONG.
  • CPXXsetlpcallbackfunc and CPXsetlpcallbackfunc
    Modifies the user-written legacy callback routine to be called after each iteration during the optimization of a linear program, and also periodically during the CPLEX presolve algorithm.
  • CPXXsetmipcallbackfunc and CPXsetmipcallbackfunc
    The routine CPXXsetmipcallbackfunc/CPXsetmipcallbackfunc sets the user-written callback routine to be called in the branch-and-cut tree, including the root node, during the optimization of a mixed integer program and during some cut generation routines.
  • CPXXsetnetcallbackfunc and CPXsetnetcallbackfunc
    Sets the user-written legacy callback routine to be called each time a log message is issued during the optimization of a network program.
  • CPXXsetnodecallbackfunc and CPXsetnodecallbackfunc
    The routine CPXXsetnodecallbackfunc/CPXsetnodecallbackfunc sets and modifies the user-written callback to be called during MIP optimization after CPLEX has selected a node to explore, but before this exploration is carried out.
  • CPXXsetnumobjs and CPXsetnumobjs
    The routine CPXXsetnumobjs/CPXsetnumobjs sets the number of objectives in the problem instance.
  • CPXXsetsolvecallbackfunc and CPXsetsolvecallbackfunc
    The routine CPXXsetsolvecallbackfunc/CPXsetsolvecallbackfunc sets and modifies the user-written callback to be called during MIP optimization to optimize subproblems (for example, node and heuristic subproblems).
  • CPXXsetstrparam and CPXsetstrparam
    The routine CPXXsetstrparam/CPXsetstrparam sets the value of a CPLEX string parameter.
  • CPXXsetterminate and CPXsetterminate
    This routine enables applications to terminate CPLEX gracefully.
  • CPXXsettuningcallbackfunc and CPXsettuningcallbackfunc
    Modifies the user-written legacy callback function to be called before each trial run during the tuning process.
  • CPXXsetusercutcallbackfunc and CPXsetusercutcallbackfunc
    The routine CPXXsetusercutcallbackfunc/CPXsetusercutcallbackfunc sets and modifies the user-written callback for adding cuts.
  • CPXXsiftopt and CPXsiftopt
    This routine looks at a subset of the columns of a problem and uses either the barrier or simplex optimizer to solve this reduced model.
  • CPXXslackfromx and CPXslackfromx
    The routine CPXXslackfromx/CPXslackfromx computes an array of slack values from primal solution values.
  • CPXXsolninfo and CPXsolninfo
  • CPXXsolution and CPXsolution
    The routine CPXXsolution/CPXsolution accesses the solution values produced by all the optimization routines except the routine CPXXNETprimopt/CPXNETprimopt.
  • CPXXsolwrite and CPXsolwrite
    The routine CPXXsolwrite/CPXsolwrite writes a solution file for the selected CPLEX problem object.
  • CPXXsolwritesolnpool and CPXsolwritesolnpool
    The routine CPXXsolwritesolnpool/CPXsolwritesolnpool writes a solution file, using either the incumbent solution or a solution from the solution pool, for the selected CPLEX problem object.
  • CPXXsolwritesolnpoolall and CPXsolwritesolnpoolall
    The routine CPXXsolwritesolnpoolall/CPXsolwritesolnpoolall writes all the solutions in the solution pool to a file for the selected CPLEX problem object.
  • CPXXstrongbranch and CPXstrongbranch
    The routine CPXXstrongbranch/CPXstrongbranch computes information for selecting a branching variable in an integer-programming branch-and-cut search.
