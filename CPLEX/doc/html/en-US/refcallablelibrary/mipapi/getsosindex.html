CPXXgetsosindex and CPXgetsosindex

The routine CPXXgetsosindex/CPXgetsosindex searches for the index number of the specified special ordered set in a CPLEX problem object.

int  CPXXgetsosindex( CPXCENVptr env, CPXCLPptr lp, char const * lname_str, CPXDIM * index_p )

int  CPXgetsosindex( CPXCENVptr env, CPXCLPptr lp, char const * lname_str, int * index_p )

Description

The routine CPXXgetsosindex/CPXgetsosindex searches for the index number of the specified special ordered set in a CPLEX problem object.

Arguments

env
A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp
A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.
lname_str
A special ordered set name to search for.
index_p
A pointer to an integer to hold the index number of the special ordered set with name lname_str. If the routine is successful, *index_p contains the index number; otherwise, *index_p is undefined.

Return

The routine returns 0 (zero) on success and nonzero if an error occurs.

Example


 status = CPXgetsosindex (env, lp, "set5", &setindex);
