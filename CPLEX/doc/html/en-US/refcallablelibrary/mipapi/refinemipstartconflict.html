int  CPXrefinemipstartconflict( CPXCENVptr env, CPXLPptr lp, int mipstartindex, int * confnumrows_p, int * confnumcols_p )

Description

This routine is deprecated. Use CPXXrefinemipstartconflictext and CPXrefinemipstartconflictext instead.

Arguments

env
A pointer to the CPLEX environment as returned by the routine CPXXopenCPLEX/CPXopenCPLEX.
lp
A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.
mipstartindex
The index of the MIP start among all the MIP starts associated with the problem.
confnumrows_p
A pointer to an integer where the number of linear constraints in the conflict is returned.
confnumcols_p
A pointer to an integer where the number of variable bounds in the conflict is returned.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.
