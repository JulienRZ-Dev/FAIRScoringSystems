CPXERR_NOT_FOR_DISTMIP

1071 problem not compatible with distributed MIP

#define CPXERR_NOT_FOR_DISTMIP 1071
Distributed MIP algorithm is incompatible with the problem at hand.
Check the problem and settings at hand to make certain that everything is compatible with the distributed MIP algorithm:
  • problem instance must be a MIP
  • VMC configuration must be specified
  • benders decomposition algorithm must not be enabled/requested
  • there can be no multiple objectives
  • there can be no legacy callbacks
For more about each of those features, see its topic in the CPLEX User's Manual.
