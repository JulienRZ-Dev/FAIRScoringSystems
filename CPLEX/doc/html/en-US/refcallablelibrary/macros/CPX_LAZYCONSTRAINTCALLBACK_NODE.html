CPX_LAZYCONSTRAINTCALLBACK_NODE

#define CPX_LAZYCONSTRAINTCALLBACK_NODE

This is the value returned by CPXXgetcallbacknodeinfo/CPXgetcallbacknodeinfo when querying CPX_CALLBACK_INFO_LAZY_SOURCE and the lazy constraint callback was invoked for an integral node.

The value is also returned by CPXXcallbackgetinfoint/CPXcallbackgetinfoint for CPXCALLBACKINFO_CANDIDATE_SOURCE in case the CANDIDATE context was invoked for a solution that originated from an integer feasible node.
