CPX_LAZYCONSTRAINTCALLBACK_MIPSTART

#define CPX_LAZYCONSTRAINTCALLBACK_MIPSTART

This is the value returned by CPXXgetcallbacknodeinfo/CPXgetcallbacknodeinfo when querying CPX_CALLBACK_INFO_LAZY_SOURCE and the lazy constraint callback was invoked from MIP start processing.

The value is also returned by CPXXcallbackgetinfoint/CPXcallbackgetinfoint for CPXCALLBACKINFO_CANDIDATE_SOURCE in case the CANDIDATE context was invoked for a solution that originated from a MIP start.

Note that in this case not all callback functionality is available since this happens very early in the search process. For example, you cannot attach or get user data to/from the current node, nor can you do anything else that requires a search tree node context.
