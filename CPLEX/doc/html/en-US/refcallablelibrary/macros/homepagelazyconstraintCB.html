Lazy constraint and callback symbols in the CPLEX Callable Library (C API)

These symbols represent information that a callback can return when the source of the callback is a lazy constraint.

  • CPX_LAZYCONSTRAINTCALLBACK_HEUR
  • CPX_LAZYCONSTRAINTCALLBACK_MIPSTART
  • CPX_LAZYCONSTRAINTCALLBACK_NODE
  • CPX_LAZYCONSTRAINTCALLBACK_USER
