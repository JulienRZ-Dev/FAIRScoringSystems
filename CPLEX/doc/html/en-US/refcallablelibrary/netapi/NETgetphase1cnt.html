CPXXNETgetphase1cnt and CPXNETgetphase1cnt

The routine CPXXNETgetphase1cnt/CPXNETgetphase1cnt returns the number of phase 1 network simplex iterations for the most recent call to CPXXNETprimopt/CPXNETprimopt.

CPXCNT  CPXXNETgetphase1cnt( CPXCENVptr env, CPXCNETptr net )

int  CPXNETgetphase1cnt( CPXCENVptr env, CPXCNETptr net )

Description

The routine CPXXNETgetphase1cnt/CPXNETgetphase1cnt returns the number of phase 1 network simplex iterations for the most recent call to CPXXNETprimopt/CPXNETprimopt.

Arguments

env
A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
net
A pointer to a CPLEX network problem object as returned by CPXXNETcreateprob/CPXNETcreateprob.

Return

Returns the total number of phase 1 network simplex iterations for the last call to CPXXNETprimopt/CPXNETprimopt, for a CPXNETptr object. If CPXXNETprimopt/CPXNETprimopt has not been called, zero is returned. If an error occurs, -1 is returned and an error message is issued.

Example


 phase1cnt = CPXNETgetphase1cnt (env, net);
