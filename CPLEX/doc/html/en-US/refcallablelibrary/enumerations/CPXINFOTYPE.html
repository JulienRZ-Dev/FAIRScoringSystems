CPXINFOTYPE

Enumeration of types of information.


typedef enum {
   CPXINFO_BYTE,
   CPXINFO_SHORT,
   CPXINFO_INT,
   CPXINFO_LONG,
   CPXINFO_DOUBLE
} CPXINFOTYPE;

This type definition enumerates the various types of information that CPXINFOHANDLER can handle and that the following routines can send:

  • CPXXsendinfobyte/CPXsendinfobyte

  • CPXXsendinfodouble/CPXsendinfodouble

  • CPXXsendinfoint/CPXsendinfoint

  • CPXXsendinfolong/CPXsendinfolong

  • CPXXsendinfoshort/CPXsendinfoshort

Name Description
CPXINFO_BYTE 8-bit signed integer.
CPXINFO_SHORT 16-bit signed integer.
CPXINFO_INT 32-bit signed integer.
CPXINFO_LONG 64-bit signed integer.
CPXINFO_DOUBLE IEEE double-precision number.
