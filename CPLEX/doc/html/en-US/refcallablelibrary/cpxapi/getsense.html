CPXXgetsense and CPXgetsense

The routine CPXXgetsense/CPXgetsense accesses the sense for a range of constraints in a CPLEX problem object.

int  CPXXgetsense( CPXCENVptr env, CPXCLPptr lp, char * sense, CPXDIM begin, CPXDIM end )

int  CPXgetsense( CPXCENVptr env, CPXCLPptr lp, char * sense, int begin, int end )

Description

The routine CPXXgetsense/CPXgetsense accesses the sense for a range of constraints in a CPLEX problem object. The beginning and end of the range must be specified.

Arguments

env
A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp
A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.
sense
An array where the specified constraint senses are to be returned. This array must be of length at least (end-begin+1). The sense of constraint i is returned in sense[i-begin]. Possible values appear in the table.
begin
An integer specifying the beginning of the range of constraint senses to be returned.
end
An integer specifying the end of the range of constraint senses to be returned.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.

Example


status = CPXgetsense (env, lp, sense, 0, cur_numrows-1);
Table 1. Values of sense
sense[i] 'L' <= constraint
sense[i] 'E' = constraint
sense[i] 'G' >= constraint
sense[i] 'R' ranged constraint
