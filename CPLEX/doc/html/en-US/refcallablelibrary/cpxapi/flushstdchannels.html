CPXXflushstdchannels and CPXflushstdchannels

The routine CPXXflushstdchannels/CPXflushstdchannels flushes the output buffers of the four standard channels cpxresults, cpxwarning, cpxerror, and cpxlog.

int  CPXXflushstdchannels( CPXCENVptr env )

int  CPXflushstdchannels( CPXCENVptr env )

Description

The routine CPXXflushstdchannels/CPXflushstdchannels flushes the output buffers of the four standard channels cpxresults, cpxwarning, cpxerror, and cpxlog. Use this routine where it is important to see all of the output created by CPLEX either on the screen or in a disk file without calling CPXXflushchannel/CPXflushchannel for each of the four channels.

Arguments

env
A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.

Example


status = CPXflushstdchannels (env);
