int  CPXXreadcopysol( CPXCENVptr env, CPXLPptr lp, char const * filename_str )

int  CPXreadcopysol( CPXCENVptr env, CPXLPptr lp, char const * filename_str )

Description

This function is deprecated. Use CPXXreadcopystartinfo and CPXreadcopystartinfo instead.

Arguments

env
A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp
A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.
filename_str
The name of the file from which the solution information should be read.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.

Example


status = CPXreadcopysol (env, lp, "myprob.sol");
