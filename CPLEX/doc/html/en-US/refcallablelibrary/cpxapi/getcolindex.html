CPXXgetcolindex and CPXgetcolindex

The routine CPXXgetcolindex/CPXgetcolindex searches for the index number of the specified column in a CPLEX problem object.

int  CPXXgetcolindex( CPXCENVptr env, CPXCLPptr lp, char const * lname_str, CPXDIM * index_p )

int  CPXgetcolindex( CPXCENVptr env, CPXCLPptr lp, char const * lname_str, int * index_p )

Description

The routine CPXXgetcolindex/CPXgetcolindex searches for the index number of the specified column in a CPLEX problem object.

Arguments

env
A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp
A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.
lname_str
A column name to search for.
index_p
A pointer to an integer to hold the index number of the column with name lname_str. If the routine is successful, *index_p contains the index number; otherwise, *index_p is undefined.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.

Example


status = CPXgetcolindex (env, lp, "power43", &colindex);
