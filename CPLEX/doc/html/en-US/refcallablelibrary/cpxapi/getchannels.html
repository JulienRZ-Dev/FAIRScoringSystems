CPXXgetchannels and CPXgetchannels

The routine CPXXgetchannels/CPXgetchannels obtains pointers to the four default channels created when you call the routine CPXXopenCPLEX/CPXopenCPLEX.

int  CPXXgetchannels( CPXCENVptr env, CPXCHANNELptr * cpxresults_p, CPXCHANNELptr * cpxwarning_p, CPXCHANNELptr * cpxerror_p, CPXCHANNELptr * cpxlog_p )

int  CPXgetchannels( CPXCENVptr env, CPXCHANNELptr * cpxresults_p, CPXCHANNELptr * cpxwarning_p, CPXCHANNELptr * cpxerror_p, CPXCHANNELptr * cpxlog_p )

Description

The routine CPXXgetchannels/CPXgetchannels obtains pointers to the four default channels created when you call the routine CPXXopenCPLEX/CPXopenCPLEX. To manipulate the messages for any of these channels, you must call this routine.

For general information about message channels and log files in CPLEX, see the topic Controlling message channels in the CPLEX User's Manual.

Arguments

env
A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
cpxresults_p
A pointer to a variable of type CPXCHANNELptr to hold the address of the channel corresponding to cpxresults. May be NULL.
cpxwarning_p
A pointer to a variable of type CPXCHANNELptr to hold the address of the channel corresponding to cpxwarning. May be NULL.
cpxerror_p
A pointer to a variable of type CPXCHANNELptr to hold the address of the channel corresponding to cpxerror. May be NULL.
cpxlog_p
A pointer to a variable of type CPXCHANNELptr to hold the address of the channel corresponding to cpxlog. May be NULL.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.

Example


status = CPXgetchannels (env, &cpxresults, &cpxwarning,
		  &cpxerror, &cpxlog);

See also lpex5.c in the topic Example: Callable Library message channels in the CPLEX User's Manual.
