CPXXgetstrparam and CPXgetstrparam

The routine CPXXgetstrparam/CPXgetstrparam obtains the current value of a CPLEX string parameter.

int  CPXXgetstrparam( CPXCENVptr env, int whichparam, char * value_str )

int  CPXgetstrparam( CPXCENVptr env, int whichparam, char * value_str )

Description

The routine CPXXgetstrparam/CPXgetstrparam obtains the current value of a CPLEX string parameter.

Arguments

env
A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
whichparam
The symbolic constant (or reference number) of the parameter for which the value is to be obtained.
value_str
A pointer to a buffer of length at least CPX_STR_PARAM_MAX to hold the current value of the CPLEX parameter.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.

Example


status = CPXgetstrparam (env, CPXPARAM_WorkDir, dirname);
