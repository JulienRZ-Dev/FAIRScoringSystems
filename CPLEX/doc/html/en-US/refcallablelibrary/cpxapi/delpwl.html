CPXXdelpwl and CPXdelpwl

The routine CPXXdelpwl/CPXdelpwl deletes a range of piecewise linear (PWL) constraints from a CPLEX problem object.

int   CPXXdelpwl( CPXCENVptr env, CPXLPptr lp, CPXDIM begin, CPXDIM end )

int   CPXdelpwl( CPXCENVptr env, CPXLPptr lp, int begin, int end )

Description

The routine CPXXdelpwl/CPXdelpwl deletes a range of piecewise linear (PWL) constraints from a CPLEX problem object. The range is specified by a lower index that represents the first PWL to be deleted and an upper index that represents the last PWL to be deleted. The indices of the constraints following those deleted constraints are automatically decreased by the number of deleted constraints.

Arguments

env
A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp
A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.
begin
An integer that specifies the numeric index of the first PWL to be deleted.
end
An integer that specifies the numeric index of the last PWL to be deleted.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.

Example


 status = CPXdelpwl (env, lp, 10, 20);
