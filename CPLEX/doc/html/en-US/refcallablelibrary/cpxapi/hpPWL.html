Piecewise linearity (PWL) in models in the CPLEX Callable Library (C API)

These routines are available in the CPLEX Callable Library (C API) to support the expression of modeling elements as piecewise linear (PWL) functions.

  • CPXXaddpwl and CPXaddpwl
    The routine CPXXaddpwl/CPXaddpwl adds a piecewise linear constraint to the specified CPLEX problem object.
  • CPXXdelpwl and CPXdelpwl
    The routine CPXXdelpwl/CPXdelpwl deletes a range of piecewise linear (PWL) constraints from a CPLEX problem object.
  • CPXXdelsetpwl and CPXdelsetpwl
    The routine CPXXdelsetpwl/CPXdelsetpwl deletes a group of piecewise linear (PWL) constraints from a CPLEX problem object.
  • CPXXgetnumpwl and CPXgetnumpwl
    The routine CPXXgetnumpwl/CPXgetnumpwl accesses the number of piecewise linear (PWL) constraints in a CPLEX problem object.
  • CPXXgetpwl and CPXgetpwl
    The routine CPXXgetpwl/CPXgetpwl accesses the definitions of a piecewise linear (PWL) constraint stored in a CPLEX problem object.
  • CPXXgetpwlindex and CPXgetpwlindex
    The routine CPXXgetpwlindex/CPXgetpwlindex searches for the index number of the piecewise liner (PWL) constraint in a CPLEX problem object.
  • CPXXgetpwlname and CPXgetpwlname
    The routine CPXXgetpwlname/CPXgetpwlname accesses the name of a piecewise linear (PWL) constraint of a CPLEX problem object.
