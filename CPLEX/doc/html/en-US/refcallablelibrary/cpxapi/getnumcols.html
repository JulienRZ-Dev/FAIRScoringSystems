CPXXgetnumcols and CPXgetnumcols

The routine CPXXgetnumcols/CPXgetnumcols accesses the number of columns in the constraint matrix, or equivalently, the number of variables in the CPLEX problem object.

CPXDIM  CPXXgetnumcols( CPXCENVptr env, CPXCLPptr lp )

int  CPXgetnumcols( CPXCENVptr env, CPXCLPptr lp )

Description

The routine CPXXgetnumcols/CPXgetnumcols accesses the number of columns in the constraint matrix, or equivalently, the number of variables in the CPLEX problem object.

Arguments

env
A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp
A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.

Return

If the problem object or environment does not exist, CPXXgetnumcols/CPXgetnumcols returns the value 0 (zero); otherwise, it returns the number of columns (variables).

Example


cur_numcols = CPXgetnumcols (env, lp);

See also the example lpex1.c in the CPLEX User's Manual and in the standard distribution.
