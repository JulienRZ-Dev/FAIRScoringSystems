CPXXgetnumdblannotations and CPXgetnumdblannotations

The routine CPXXgetnumdblannotations/CPXgetnumdblannotations accesses the number of double values annotations in the problem object.

CPXDIM  CPXXgetnumdblannotations( CPXCENVptr env, CPXCLPptr lp )

int  CPXgetnumdblannotations( CPXCENVptr env, CPXCLPptr lp )

Description

The routine CPXXgetnumdblannotations/CPXgetnumdblannotations accesses the number of double values annotations problem object.

Arguments

env
A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp
A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.

Return

If the CPLEX problem object or environment does not exist, CPXXgetnumdblannotations/CPXgetnumdblannotations returns the value 0 (zero); otherwise, it returns the number of annotations.

Example


cur_numannotations = CPXgetnumdblannotations (env, lp);

See also the example lpex1.c in the CPLEX User's Manual and in the standard distribution.
