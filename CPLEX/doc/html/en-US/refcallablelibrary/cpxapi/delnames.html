CPXXdelnames and CPXdelnames

The routine CPXXdelnames/CPXdelnames removes all names that have been previously assigned to model entities, such as rows, columns, quadratic constraints, special ordered sets (SOSs), or indicator constraints.

int  CPXXdelnames( CPXCENVptr env, CPXLPptr lp )

int  CPXdelnames( CPXCENVptr env, CPXLPptr lp )

Description

The routine CPXXdelnames/CPXdelnames removes all names that have been previously assigned to model entities, such as rows, columns, quadratic constraints, special ordered sets (SOSs), or indicator constraints. The memory that was used by those names is released.

Names can be assigned to rows and columns in a variety of ways, and this routine allows them to be removed. For example, when a problem is read from a file in LP or MPS format, names are also read from the file. Names can also be assigned by the user when the user invokes one of the routines CPXXchgrowname/CPXchgrowname, CPXXchgcolname/CPXchgcolname, or CPXXchgname/CPXchgname.

Arguments

env
A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp
A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.

Example


CPXdelnames (env, lp);
