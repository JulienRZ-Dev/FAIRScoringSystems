CPXXdistmipopt and CPXdistmipopt

This routine is deprecated and will be removed in a future release.

int  CPXXdistmipopt( CPXCENVptr env, CPXLPptr lp )

int  CPXdistmipopt( CPXCENVptr env, CPXLPptr lp )

Description

This routine is deprecated and will be removed in a future release.

At any time after a mixed integer program has been created by a call to CPXXcreateprob/CPXcreateprob, you can invoke the routine CPXXdistmipopt/CPXdistmipopt to find a solution to that problem using a distributed parallel algorithm.

Before you can invoke this routine, you must install a parallel machine configuration using either CPXXcopyvmconfig/CPXcopyvmconfig or CPXXreadcopyvmconfig/CPXreadcopyvmconfig. Such a machine configuration specifies which (virtual) machines the distributed memory MIP solver can use in order to optimize the problem.

Arguments

env
A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp
A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.

Example


 status = CPXdistmipopt (env, lp);
