CPXXdelvmconfig and CPXdelvmconfig

This routine is deprecated and will be removed in a future release.

int  CPXXdelvmconfig( CPXENVptr env )

int  CPXdelvmconfig( CPXENVptr env )

Description

This routine is deprecated and will be removed in a future release.

This routine deletes any parallel machine configuration that was previously loaded by CPXXcopyvmconfig/CPXcopyvmconfig or CPXXreadcopyvmconfig/CPXreadcopyvmconfig. If no parallel machine configuration was loaded, then this routine does nothing.

Arguments

env
A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.

Example


 status = CPXdelvmconfig (env);
