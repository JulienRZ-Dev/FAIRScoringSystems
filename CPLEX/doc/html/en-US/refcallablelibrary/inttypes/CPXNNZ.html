CPXNNZ

Integer type for nonzero counts.

typedef configuration_dependent_type CPXNNZ
CPXNNZ is the integer type that cplexx.h uses to pass nonzero counts to the CPLEX Callable Library (C API) or to read such data from it. The actual type of CPXNNZ depends on the current setting of CPX_APIMODEL.
