Type definitions in the CPLEX Callable Library (C API) Reference Manual

These type definitions (also known as typedefs) define symbolic values in the CPLEX Callable Library (C API).

  • CPXASYNCptr
    Pointer to an instance of cpxasynchandle.
  • CPXCALLBACKCONTEXTptr
    Pointer to a callback context.
  • CPXCALLBACKFUNC
    Prototype for generic callback functions
  • CPXCASYNCptr
    Pointer to a constant instance of cpxasynchandle.
  • CPXENVGROUPptr
    A group of environments.
  • CPXCENVGROUPptr
    A group of environments.
  • CPXSERIALIZERptr
    Pointer to a serializer.
  • CPXCSERIALIZERptr
    Pointer to a serializer.
  • CPXDESERIALIZERptr
    Pointer to a deserializer.
  • CPXCDESERIALIZERptr
    Pointer to a deserializer.
  • CPXMESSAGEHANDLERptr
    Pointer to an opaque structure representing a message handler on the remote worker.
  • CPXCMESSAGEHANDLERptr
    Pointer to an opaque structure representing a message handler on the remote worker.
  • CPXINFOHANDLER
    Prototype for functions to handle information messages on a client.
  • CPXMODELASSTCALLBACKFUNC
    Prototype for modeling assistance callback functions
  • CPXUSERFUNCTION
    Prototype for user functions that can be registered with a remote worker.
