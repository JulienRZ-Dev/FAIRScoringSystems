CPXUSERFUNCTION

Prototype for user functions that can be registered with a remote worker.

typedef int(CPXPUBLIC CPXUSERFUNCTION)( CPXENVptr env, int id, CPXLONG insize, void const* indata, CPXLONG maxout, CPXLONG *outsize_p, void * outdata, void * handle )
Name Description
env The environment on behalf of which the function is invoked.
id The function id that was passed to CPXXuserfunction/CPXuserfunction on the master.
insize The length of indata (as passed to CPXXuserfunction/CPXuserfunction on the master).
indata Input data for the function (as passed to CPXXuserfunction/CPXuserfunction on the master).
maxout Capacity of outdata.
outsize_p Here the function should store the number of bytes actually written to outdata. The value stored here will be returned by CPXXuserfunction/CPXuserfunction on the master.
outdata Buffer for output data. The capacity of this buffer is specified by maxout.
handle The handle that was specified when the function was registered.

Return

The function should return 0 (zero) on success or an error code otherwise.
