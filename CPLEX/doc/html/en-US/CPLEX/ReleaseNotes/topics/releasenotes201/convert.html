Changes in features of CPLEX 20.10

These release notes highlight changes of features in IBM CPLEX 20.1.0. Please review these notes before using IBM CPLEX 20.1.0.

Python 3.6 support dropped

Support of Python 3.6 has been dropped with this release. Upgrade your Python 3.6 code to use Python 3.7 or Python 3.8 versions.
