New features in the Java API

These release notes highlight new features in the Java API of IBM CPLEX 20.1.0. For more information, see the reference manual of the Java API of CPLEX.

New conflict refiner example

The following new example, demonstrating how to use the conflict refiner, has been added in CPLEX 20.1.0:

  • <Install_dir>/cplex/examples/java/ConflictEx1.java

New enumeration constant to report number of open nodes from generic callback

The enumeration IloCplex.Callback.Context.Info contains one new constant in CPLEX 20.1.0:

  • NodesLeft

Please see the documentation of CPXCALLBACKINFO for more information.

New method IloCplex.readStartInfo()

The new method IloCplex.readStartInfo() has been added. This method reads starting information from the SOL file denoted by name and copies this information into the invoking object.
