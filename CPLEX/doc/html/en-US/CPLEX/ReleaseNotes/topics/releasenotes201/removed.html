Removed features

These release notes announce features that were deprecated in a previous release and have been removed from IBM CPLEX V20.1.

MATLAB Connector

The MATLAB Connector has been removed from CPLEX V20.1.

Callable Library (C API)

Routines

The following routines have been removed in CPLEX V20.1:

  • CPXXmsg and CPXmsg

Instead, use CPXXmsgstr and CPXmsgstr.

The following undocumented routines have been removed in CPLEX V20.1:

  • CPXXlprewrite and CPXlprewrite
  • CPXXlpwrite and CPXlpwrite
  • CPXXmpsrewrite and CPXmpsrewrite
  • CPXXmpswrite and CPXmpswrite
  • CPXXsavwrite and CPXsavwrite

Instead, use CPXXwriteprob and CPXwriteprob.
