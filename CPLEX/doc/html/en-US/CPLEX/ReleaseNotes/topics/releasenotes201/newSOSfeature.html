New special ordered set (SOS) algorithmic feature

CPLEX 20.1.0 offers an automatic logarithmic reformulation of special ordered sets of type 1 (SOS1) and type 2 (SOS2).

New algorithmic feature for SOS1 and SOS2

CPLEX 20.1.0 offers an automatic logarithmic reformulation of special ordered sets. On problems containing special ordered sets of type 1 (SOS1) or type 2 (SOS2), this reformulation can yield a huge speed-up in the solution time. The same also applies to problems formulated with piecewise linear (PWL) functions, since PWL functions are internally reformulated with special ordered sets of type 2 (SOS2).

For more information, see Automatic reformulation of special ordered sets in the CPLEX User's Manual and the documentation for the two new parameters:

  • SOS1 reformulations (CPXPARAM_Preprocessing_SOS1Reform)
  • SOS2 reformulations (CPXPARAM_Preprocessing_SOS2Reform)
