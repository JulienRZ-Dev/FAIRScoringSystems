Deprecated features

These release notes announce deprecated features in IBM CPLEX 20.1.0. These deprecated features will be removed in a future release.

Distributed parallel optimization of mixed integer programming models (MIP)

Distributed parallel optimization of mixed integer programming (MIP) models has been deprecated in CPLEX 20.1.0 and will be removed in a future release.

Remote Object functionality

Remote Object functionality has been deprecated in CPLEX 20.1.0 and will be removed in a future release.

Parameters

The parameter CPXPARAM_Preprocessing_Linear, has been deprecated and will be removed in a future release. The new parameter PXPARAM_Preprocessing_Reformulations allows you to set presolve reformulations and replaces this functionality.

Callable Library (C API)

Routines

The following routine has been deprecated in CPLEX 20.1.0 and will be removed in a future release: CPXXrefineconflict and CPXrefineconflict. Instead, use CPXXrefineconflictext and CPXrefineconflictext.

The following routine has been deprecated in CPLEX 20.1.0 and will be removed in a future release: CPXXrefinemipstartconflict and CPXrefinemipstartconflict. Instead, use CPXXrefinemipstartconflictext and CPXrefinemipstartconflictext.

The following routine has been deprecated in CPLEX 20.1.0 and will be removed in a future release: CPXXreadcopysol and CPXreadcopysol. Instead, use CPXXreadcopystartinfo and CPXreadcopystartinfo.

C++ API

The member function IloCplex::readSolution() has been deprecated. Use the new method IloCplex::readStartInfo() instead.

Java API

The method IloCplex.readSolution() has been deprecated. Use the new method IloCplex.readStartInfo() instead.

C#/.NET API

The method Cplex.ReadSolution() has been deprecated. Use the new method Cplex.ReadStartInfo() instead.
