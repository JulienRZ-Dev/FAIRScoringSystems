New features in the C#/.NET API

These release notes highlight new features in the C#/.NET API of IBM CPLEX 20.1.0. For more information, see the reference manual of the C#/.NET API of CPLEX.

New conflict refiner example

The following new example, demonstrating how to use the conflict refiner, has been added in CPLEX 20.1.0:

  • <Install_dir>/cplex/examples/cs/ConflictEx1.cs

New enumeration member to report number of open nodes from generic callback

The enumeration Cplex.Callback.Context.Info contains one new constant in CPLEX 20.1.0:

  • NodesLeft

Please see the documentation of CPXCALLBACKINFO for more information.

New method Cplex.ReadStartInfo()

The new method Cplex.ReadStartInfo() has been added. This method reads starting information from the SOL file denoted by name and copies this information into the invoking object.
