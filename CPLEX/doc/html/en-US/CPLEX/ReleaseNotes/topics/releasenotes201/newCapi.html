New features in the Callable Library (C API)

These release notes highlight new features in the Callable Library (C API) of IBM CPLEX 20.1.0.

New routines in conflict refiner

The following new routines have been added in CPLEX 20.1.0:
  • CPXXgetconflictgroups and CPXgetconflictgroups
  • CPXXgetconflictnumgroups and CPXgetconflictnumgroups
  • CPXXgetconflictnumpasses and CPXgetconflictnumpasses

New conflict refiner examples

The following new examples, demonstrating how to use the conflict refiner, have been added in CPLEX 20.1.0:

  • <Install_dir>/cplex/examples/src/c/conflictex1.c
  • <Install_dir>/cplex/examples/src/c_x/xconflictex1.c

New routine to read start information a SOL format file

The new routine CPXXreadcopystartinfo and CPXreadcopystartinfo reads start information from a SOL format file, and copies that basis or solution into a CPLEX problem object.

New enumeration value to report number of open nodes from generic callback

The enumeration CPXCALLBACKINFO contains one new value in CPLEX 20.1.0:

  • CPXCALLBACKINFO_NODESLEFT

Please see the documentation of CPXCALLBACKINFO for more information.
