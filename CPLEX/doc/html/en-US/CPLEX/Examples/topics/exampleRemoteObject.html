Examples of the Distributed Parallel MIP

These examples in C, C++, C#.NET, and Python show how to develop an application of distributed parallel mixed integer programming (MIP).

Note:

Distributed parallel optimization of mixed integer programming (MIP) models has been deprecated in CPLEX 20.1.0 and will be removed in a future release.

The following examples are delivered with CPLEX in IBM ILOG CPLEX Optimization Studio. They illustrate how to develop a MIP optimization distributed among multiple machines to run in parallel in either the C, C++, Java, C#.NET, or Python programming language.

xdistmipex1.c runs MIP optimization distributed in parallel.
xdistmipex2.c runs MIP optimization distributed in parallel and applies an informational callback.
ilodistmipex1.cpp runs MIP optimization distributed in parallel.
ilodistmipex2.cpp runs MIP optimization distributed in parallel and applies an informational callback.
DistMIPEx1.java runs MIP optimization distributed in parallel.
DistMIPEx1.java runs MIP optimization distributed in parallel and applies an informational callback.
distmipex1.py runs MIP optimization distributed in parallel.
distmipex2.py runs MIP optimization distributed in parallel and applies an informational callback.
DistMIPEx1.cs runs MIP optimization distributed in parallel. This example is available only on Windows platforms.
DistMIPEx2.cs runs MIP optimization distributed in parallel and applies an informational callback. This example is available only on Windows platforms.
