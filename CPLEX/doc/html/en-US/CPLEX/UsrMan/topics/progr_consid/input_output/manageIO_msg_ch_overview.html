Overview

Introduces message channels for CPLEX.

In both the Interactive Optimizer and the Callable Library, there are message channels that enable you to direct output from your application as you prefer. In the Interactive Optimizer, these channels are defined by the command set output channel with its options as listed in Table 1.

In the Python API, the class Cplex provides the methods set_results_stream, set_warning_stream, set_error_stream, and set_log_stream to control output channels.

In the Callable Library, there are routines for managing message channels, in addition to parameters that you can set. In the C++ and Java APIs, the class IloCplex inherits methods from the Concert Technology class IloAlgorithm , methods that enable you to control input and output channels.
