Examples for multiobjective optimization

Provides an overview of the examples using multiobjective optimization.

The multiobjex1 example demonstrates how to read in and optimize a multiobjective problem. It also demonstrates how to use parameter sets with multiobjective optimization.

  • examples/src/java/MultiObjEx1.java
  • examples/src/cs/MultiObjEx1.cs
  • examples/src/python/multiobjex1.py

The diet example has been modified to demonstrate how to model a multiobjective problem by optionally creating a second objective. It also demonstrates how to use parameter sets with multiobjective optimization.
