DOcplex examples

DOcplex examples are available for constraint programming and mathematical programming, either as Jupyter Notebooks or Python models.

Tutorials

These tutorials are presented as Jupyter Notebooks, so you can work through them at your own pace.

For mathematical programming, you can find two introductory tutorials on the Decision Optimization GitHub:

  • Linear Programming, (CPLEX Part 1)
  • Beyond Linear Programming, (CPLEX Part 2)

For constraint programming, you can find an introductory tutorial on the Decision Optimization GitHub:

  • Getting started with Scheduling in DOcplex

Other Jupyter Notebooks for MP

Other Notebooks for mathematical programming include:

  • Benders decomposition
  • Using progress callbacks
  • Finding optimal locations of new stores
  • Incremental modeling
  • Optimizing mining operations

Other Jupyter Notebooks for CP

Other Notebooks for constraint programming include:

  • Scheduling games for a sports league
  • Housebuilding with differently skilled workers
  • Steel mill slab design
  • The eight queens puzzle
  • Optimizing truck configuration for deliveries
